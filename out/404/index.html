Sobre el proyecto

Este proyecto fue desarrollado por mí, Lilith Díaz, para el programa "Riesgos Catastróficos Globales" de la organización Carreras con Impacto.

Para ver detalles técnicos y acceder al write-up consultar el repositorio en Github del proyecto.

Objetivos del proyecto

Este proyecto pretende atender el área de riesgos biológicos mediante la creación de un repositorio que almacene información de importancia epidemiológica de los últimos 25 años en México (2000-2024).

La recolección de estos datos ayudará al futuro análisis de los eventos con el fin de no solamente facilitar la respuesta de México ante estos riesgos, sino de otros países; la comparación de respuestas facilitará la creación de estrategias modificadas de acuerdo a casos particulares.
